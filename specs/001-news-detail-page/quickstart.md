# Quickstart: News Detail Page

## Prerequisites
- [x] Base React project initialized.
- [ ] Install routing: `npm install react-router-dom`

## Setup & Development

### 1. Run Development Server
```bash
npm run dev
```

### 2. Verify Home Page Links
- Open `http://localhost:5173`.
- Click on any news card or carousel item.
- Ensure the URL changes to `/news/:id`.

### 3. Direct Access
- Navigate directly to `http://localhost:5173/news/1`.
- Verify that the article "A Reconstrução Urbana no Século XXI" is displayed with its full content.

## Testing
Run the test suite to verify routing and component integrity:
```bash
npm run test
```
